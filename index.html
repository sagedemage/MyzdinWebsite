About

A simple game written in C++ using SDL2. It is supposed to be a 2D platformer. I am trying to make this game portable to many operating systems (Windows, MacOS, Linux and etc..). I can only support this game for Windows and Linux. This game is alpha software. There are many features missing for it to be a complete game.

My Music

I wrote the music for this game. Since it is an open source project. You can freely use, download, and modify music scores. I do not care if you use unmodified copies of my music. The only thing I want is to make sure you give credit to the music I wrote.

Controls

  • A -> move left
  • D -> move left
  • K -> jump

Closing

I hope to do more than this. This is my first time programming with SDL2. I have used C++ before. I am determined to get a fully functional game but it will take some time.
