Build the game yourself in Linux

package requirements

Debian and Debian based distros

  • g++ (GNU GCC Compiler)
  • libsdl2-2.0-0
  • libsdl2-mixer-2.0-0
  • libsdl2-image-2.0-0
  • libsdl2-dev
  • libsdl2-mixer-dev
  • libsdl2-image-dev
  • cmake
  • clang-tools-7 (optional)

RHEL and RHEL based distros

  • gcc-c++ (GNU GCC Compiler)
  • SDL2
  • SDL2_mixer
  • SDL2_image
  • SDL2-devel
  • SDL2_mixer-devel
  • SDL2_image-devel
  • cmake
  • clang-tools-extra (optional)

Build using CMake

$ mkdir build
$ cmake -B build
$ cd build
$ cmake --build .

Use clang-tidy to lint my program

$ mkdir script-tidy
$ cd script-tidy
$ cmake ../ -DCMAKE_EXPORT_COMPILE_COMMANDS=ON
$ clang-tidy -p ./ ../src/[cpp file path] -checks=*,-modernize*,-llvmlibc*,-llvm*,-hicpp* -header-filter=.*
Example:
$ clang-tidy -p ./ ../src/main.cpp -checks=*,-modernize*,-llvmlibc*,-llvm*,-hicpp* -header-filter=.*
